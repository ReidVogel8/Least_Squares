# Least_Squares
In this repository we are using different methods of least squares to find the line of best fit. Getting a data set regarding the correlation between students time spent studying and their exam grades. I am proving my hypothesis 'There is a positive correlation between students time studying and obtaining a higher exam score.'

Least_Squares_Report.pdf: This is pdf report of my findings throught this research. Here I explain how and why I am getting the results.

least_squares.py: This is the python file (In the appendix of Least_Squares_Report.pdf) using different methods of least squares to find the line of best fit for the data set.
